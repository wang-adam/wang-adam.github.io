Projects
BMTron

Two-player game where the goal is to make the opponent crash into the edge or a previous crossed path. Modeled after a game I used to play when when I was little.

Conway's Game of Life

Created the Conway's cellular automaton using React. "The Game of Life is undecidable, which means that given an initial pattern and a later pattern, no algorithm exists that can tell whether the later pattern is ever going to appear." Play with it here!

Brownian Motion Visual

Simulates the random motion of particles in liquids and gasses. For example, pollen movement in water due to the collision of the water molecules. Further reading.

Clothing Neural Network

Built a Neural Network using TensorFlow that categorizes types of clothing. I adapted it to predict types of clothes from the internet.

Facial Recognition and Foreground Processing

Uses the OpenCV Python library to find the face and eyes of people in images. Also uses the library to extract foregrounds of an image.

Moon Tracker Web Application

Calculates the ideal spot to stand so that the moon appears behind a landmark using input data and lunar cycle. My CSE 143 final project with Kavel Rao and Alek Metzelaar.

Sierpinski's Triangle Visualization

Generates Sierpinksi's Triangle using chaos game: with a random point inside the triangle and a random corner, draw the midpoint. Repeat using the midpoint.

Mine Sweeper

Built the classic Mine Sweeper game as a web application using React. Mines are randomly placed onto the board and board size can be adjusted. Play it here!

Monte Carlo Simulation of S&P500

Uses the Monte Carlo Simulation to generate projections of S&P500 using the drift and a random factor. Data is parsed from an .csv file downloaded from Yahoo Finance.

Approximating e by Simulation

Approximating e using a simple simulation (Simulation 1), and a more complex one that uses antithesis variables to reduce data variance (Simulation 2). Based off "Estimating the Value of e by Simulation" by K. G. Russell.

TicTacToe with React

Has PvP or PvC game modes. The computer has a "dummy" difficulty that selects a random square and a "smart" difficulty that plays with optimal strategy. Built with Java in high school and later built it into a React web application.

Recursive Binary Tree Visualization

Accepts input for number of levels and angle between each branch. Interesting how placement of the draw statement inside the recursion affects the visualization.

Best Day of the Week to Trade Stocks

Finds which day of the week is the best to buy/sell. Calculates the average percent gain with respect to Monday's closing price.

Random vs Seemingly Random Points

Explores the concept of truly random points versus points that seem random but are not. Inspired by How Not to Be Wrong: The Power of Mathematical Thinking by Jordan Ellenberg.

Koch Snowflake Visualization

Shows a few iterations of the Koch Snowflake. It's interesting how as the number of iterations reach infinity, the perimeter approaches infinite but the area remains finite.

Wellness Notification

Sends a desktop reminder every hour to drink water and take a screen break.

Bouncing Circles Screensaver

Circle collisions inelastically and form a new circle that follows the Law of Conservation of Momentum in physics.

Pong Arcade Game

Classic player vs. computer Pong game. The computer (right side) only moves when the ball is on their side, giving the player a slight advantage when the ball speeds up.

TextExcel Spreadsheet

Functions as a basic Excel spreadsheet. It can perform arithmetic operations and find the sum and average of a group of cells. It also deals with overflow content. Was a project for AP Computer Science.
